# React Availability Calendar

A web application to manage and view user availability slots using [react-big-calendar](https://github.com/jquense/react-big-calendar) with a custom dialog implementation, RESTful backend integration, and enhanced UX features like slot validation, conflict resolution, and custom deletion confirmation.
